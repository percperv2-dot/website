# Onion AI Bot - Telegram Bot

Bot Telegram semplice che risponde al comando `/start` con il link al sito web.

## Installazione

1. Assicurati di avere Python 3.8 o superiore installato:
   ```powershell
   python --version
   ```

2. Installa le dipendenze:
   ```powershell
   pip install -r requirements.txt
   ```

## Avvio del Bot

Su PowerShell, esegui:
```powershell
python bot.py
```

Il bot si avvierà e inizierà a rispondere ai comandi.

## Comandi Disponibili

- `/start` - Mostra il messaggio di benvenuto con il link www.oaibot.net
- `/help` - Mostra i comandi disponibili

## Fermare il Bot

Premi `Ctrl+C` nella finestra PowerShell per fermare il bot.

## Note

- Il bot deve essere sempre in esecuzione per rispondere ai messaggi
- Assicurati di avere una connessione internet attiva
- Il token del bot è già configurato nel file `bot.py`
